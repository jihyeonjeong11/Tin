## 주식 연동

놓아버림 Tin에 주식 관련 기록이 있을 경우, "그때 샀으면 지금은 얼마?" 를 보여주는 기능.

- 예: "애플 주식 살까 하다가 포기함" → 그 날짜 기준으로 현재까지 수익률 표시
- 주가 API 연동 필요 — Alpha Vantage(무료 분당 5회/일 500회) 또는 Finnhub(무료 분당 60회) 검토. 과거 특정 날짜 종가만 필요하므로 무료 티어로 충분할 것으로 예상
- Tin 작성 시 주식 티커 태깅 옵션 추가
- "그때 $1,000 투자했으면 지금 $3,240" 식의 카드 UI
